JavaScript

Javascript often abbreivated as JS, is a high-level, just-in-tume complied, object-oriented programming language that conforms to the ECMAscript specification, JavaScript has curly-bracket syntax, dynamic typing, prototype-based object-orientation, and first-class functions.

Alongside HTML and CSS, javaScript is one of the core technologies of the Worle Wide Web. JavaScript enables interactive web pages and is an essential part of web applications. The vast majority of websites use it, and major web browsers have a dedicated JavaScript engine to execute it.
